You are a senior full-stack developer and UI/UX designer.

Build a modern, high-converting travel & car rental website for an Indian travel company.

========================
🧠 BUSINESS OVERVIEW
========================
The company provides chauffeur-driven vehicles for tours across India:
- Himachal Pradesh
- Uttarakhand
- Rajasthan
- Kashmir
- Kerala
- Mumbai
- Goa
- Gujarat

Fleet includes:
- Innova Crysta (premium SUV)
- Tempo Traveller (group travel)
- Swift Dzire (budget sedan)
- Honda Amaze
- Ertiga

Core services:
- Outstation tours
- Custom travel packages
- Airport transfers
- Group tours
- Corporate bookings

========================
🎯 GOAL
========================
Create a HIGH-CONVERSION website similar to:
- Savaari (simple booking UX, multi-service offering)
- Avis India (premium feel, trust-based UI)
- Silverline Travels (fleet + tour focus)

The website must:
- Build trust instantly
- Showcase fleet clearly
- Drive inquiries & bookings
- Look premium but Indian-market friendly

========================
🎨 DESIGN SYSTEM (VERY IMPORTANT)
========================
Use design inspiration from:
https://seafarer.qodeinteractive.com/

Theme guidelines:
- Colors:
  - Primary: Deep Navy Blue (#0D1B2A)
  - Accent: Ocean Blue (#1B6CA8)
  - Highlight: Gold (#D4AF37)
  - Background: Soft white (#F8F9FA)
- Fonts:
  - Headings: Playfair Display (luxury feel)
  - Body: Poppins or Open Sans
- Style:
  - Minimal + premium
  - Large hero sections
  - Smooth animations
  - Clean spacing
  - Card-based layouts
  - Parallax scrolling

========================
📄 WEBSITE STRUCTURE
========================

1. HOME PAGE
- Hero section with:
  - Background travel video/image
  - Heading: “Explore India with Premium Chauffeur Services”
  - Booking form (pickup, drop, date, car type)
- USP section:
  - Verified drivers
  - Well-maintained vehicles
  - Transparent pricing
  - 24/7 support
- Fleet preview (cards)
- Popular destinations (cards with images)
- Testimonials
- CTA banner

2. FLEET PAGE
- Filterable vehicle categories:
  - Sedan
  - SUV
  - Tempo Traveller
  - Bus
- Each vehicle card:
  - Image
  - Seating capacity
  - Features
  - Ideal for
  - “Book Now” button

3. DESTINATIONS PAGE
- Grid of destinations:
  - Himachal, Uttarakhand, Rajasthan, Kashmir, Kerala, Goa, etc.
- Each opens:
  - Tour highlights
  - Suggested itinerary
  - Vehicle recommendations

4. SERVICES PAGE
- Outstation trips
- Airport transfers
- Corporate travel
- Group tours
- Wedding transport

5. ABOUT PAGE
- Company story
- Why choose us
- Stats (trips completed, cities covered)

6. CONTACT PAGE
- Inquiry form
- WhatsApp button
- Phone/email
